# zen-cli

>A simple CLI for creating my projects.

## setup

### 安装

```bash
windows

$ npm i @jomsou/cli -g

mac

$ sudo i npm @jomsou/cli -g
```

### 增加模板

```bash
$ sudo zen add 
```

### 模板列表

```bash
$ zen list
```

### 初始化项目

```bash
$ zen init 
```

